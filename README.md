# 在线学习系统简介 python98

## 项目概述


## 技术栈

- **后端**: Python
- **Web框架**: Django
- **数据库**: MySQL
- **开发工具**: PyCharm

## 功能模块

- 用户管理：注册、登录、信息修改等
- 课程管理：课程展示、课程详情、课程学习等
- 报告生成：学习进度报告自动生成

## 系统角色

- **学生**: 可以浏览课程、学习课程、查看学习报告等
- **管理员**: 负责课程内容的管理和用户管理

## 运行环境

- 操作系统: 兼容主流操作系统
- Python版本: 3.x
- 数据库: MySQL 5.x 或更高版本

## 部署步骤

1. 确保Python环境和MySQL数据库已正确安装
2. 克隆或下载项目代码
3. 安装依赖库（通过pip）
4. 配置数据库连接信息
5. 运行迁移以创建数据库表
6. 启动Django开发服务器
